NAME
====

Units 

SYNOPSIS
========

    use Units <m>;
    say 8A * 2Ω; 
    
    # Output: 8V

DESCRIPTION
===========

Implements all SI and a couple of derived Units as Operators. Units can be converted and calulated with.
Please see the tests.
